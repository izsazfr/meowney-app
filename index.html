Cara setup Apps Script
  1. Buka Google Sheet kamu → Extensions → Apps Script
  2. Hapus isi default, tempel kode dari file Code.gs yang Claude berikan
  3. Klik Deploy → New deployment → pilih tipe Web app
  4. Execute as: Me. Who has access: Anyone
  5. Klik Deploy, salin Web app URL, tempel di kolom atas
  6. Kalau update kode Code.gs lagi nanti, pakai Deploy → Manage deployments → edit → New version
